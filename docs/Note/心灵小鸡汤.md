**“这世界能人辈出，天才俊杰比比皆是。漫漫历史长河中，到处都是闪烁着的明星巅峰。要想脱颖而出、勇猛精进，唯有利用前贤遗藏和积累，将他们的精华都吸收到到自己的身上，正所谓海纳百川，有容乃大，采百家之众长，方可领袖群雄，乃至凌驾众生之上，达成无上伟绩！”
